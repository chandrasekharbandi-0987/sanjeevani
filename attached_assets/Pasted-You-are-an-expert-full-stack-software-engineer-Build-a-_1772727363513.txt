You are an expert full-stack software engineer. Build a complete production-ready web application called Sanjeevani, a modern healthcare platform that connects blood donors, recipients, doctors, and hospitals to save lives during emergencies. The application must be user-friendly, secure, scalable, and visually inspiring so users feel motivated to donate blood.

The system must include modern UI design, backend APIs, authentication, role-based dashboards, donor search, emergency alerts, donation tracking, notifications, and real-time communication.

The design should feel like a professional humanitarian medical service similar to the digital platforms used by the American Red Cross.

APPLICATION PURPOSE

Sanjeevani allows:

• donors to register and offer blood donation
• recipients to request blood during emergencies
• doctors to verify donation eligibility
• hospitals to coordinate donation events
• administrators to manage the system

The goal is to reduce emergency response time and encourage voluntary blood donation.
